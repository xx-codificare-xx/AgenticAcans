## Music

The portfolio has background music that fits the dark techie vibe.

**To hear it:** click anywhere on the page once — the browser requires a user interaction before playing audio, this is a Chrome/Safari/Firefox rule and not something that can be bypassed.

**To stop it:** click the **♪ MUSIC ON** button in the top right corner of the navigation bar. Click again to resume.

## Running Locally

No install needed. Two options:

**Option 1 — Live Server (recommended)**
1. Open the folder in VSCode
2. Install the Live Server extension (Ritwick Dey)
3. Right-click `index.html` → Open with Live Server

**Option 2 — Direct**
Double-click `index.html` to open in browser. Note: music won't play over `file://` protocol, use Live Server for that.
